# Baseline RAG Methods

This directory contains implementations of state-of-the-art baseline RAG methods for comparison with ARK. All baselines were cloned from their official GitHub repositories and adapted for evaluation on our benchmarks.

---

## Included Baselines

### 1. MemoRAG
**Memory-Augmented Retrieval for Long-Context Question Answering**

**Repository**: [https://github.com/qhjqhj00/MemoRAG](https://github.com/qhjqhj00/MemoRAG)
**Paper**: MemoRAG: Moving Towards Next-Gen RAG Via Memory-Inspired Knowledge Discovery
**Clone Info**:
- Branch: `main`
- Commit: `67a88db` (April 23, 2025)

---

### 2. HippoRAG
**Neurobiologically-Inspired Long-Term Memory for RAG**

**Repository**: [https://github.com/OSU-NLP-Group/HippoRAG](https://github.com/OSU-NLP-Group/HippoRAG)
**Paper**: HippoRAG: Neurobiologically Inspired Long-Term Memory for Large Language Models (EMNLP 2024)
**Clone Info**:
- Branch: `main`
- Commit: `242abfb` (June 24, 2025)

---

### 3. LightRAG
**Lightweight and Fast Graph-Based RAG**

**Repository**: [https://github.com/HKUDS/LightRAG](https://github.com/HKUDS/LightRAG)
**Paper**: LightRAG: Simple and Fast Retrieval-Augmented Generation
**Clone Info**:
- Branch: `main`
- Commit: `ba0cffd` (July 9, 2025)

---

### 4. nano-graphrag
**Community-Driven Lightweight GraphRAG Implementation**

**Repository**: [https://github.com/gusye1234/nano-graphrag](https://github.com/gusye1234/nano-graphrag)
**Paper**: A simplified, community-maintained implementation of GraphRAG concepts
**Clone Info**:
- Branch: `main`
- Commit: `6382ab6` (April 12, 2025)

**Note**: This is the community version of GraphRAG (nano-graphrag), not Microsoft's official GraphRAG implementation. We chose this version for its lightweight design and ease of integration.
